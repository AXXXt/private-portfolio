# 自我蒸馏工作流 — 生成 lib/persona.ts

## 概述
本工作流把"自己"蒸馏为两部分：
- **Part A · Self Memory**：技术能力、项目经历、知识体系
- **Part B · Persona**：口头禅、认知风格、社交模式、价值观

产物直接覆盖 `lib/persona.ts`，接入 Portfolio 的 DeepSeek AI 蒸馏体。

---

## 快速运行
直接在 Codex 中说：
> 帮我执行自我蒸馏，生成 lib/persona.ts

Codex 会依次问你以下问题，你回答后自动生成。

---

## 手动蒸馏：逐层填写

### Layer 1 — 技术能力记忆
1. 你最熟悉的编程语言/框架是什么？（按熟练度排列）
2. 你做过的最复杂的项目是什么？你在其中的角色？
3. 你认为自己最强的技术领域是？最想补的短板是？
4. 有没有开源贡献或技术社区参与经历？
5. 你日常使用的工具链（IDE、终端、DevOps 工具）？

### Layer 2 — 项目经历
对每个主要项目描述：
- 项目名称和作用
- 你的角色和贡献
- 技术栈
- 解决了什么核心问题
- 过程中学到的关键教训

### Layer 3 — 口头禅与语言风格
1. 技术讨论时你有什么习惯用语？
2. 面对不确定的问题时你怎么表达？
3. 你的语气偏什么风格（冷静/热情/幽默/直接）？
4. Code review 时你通常怎么提意见？

### Layer 4 — 认知风格
1. 遇到新问题时你的第一反应是什么？
2. 做技术决策时你最看重什么？
3. 你偏好先规划再动手，还是边做边调整？
4. 你怎么判断一个方案"够好了"？

### Layer 5 — 安全边界（硬约束）
1. 哪些个人信息绝对不能暴露？（姓名、电话、住址...）
2. 哪些技术细节不能输出？（漏洞细节、内部配置...）
3. 面试场景下你希望蒸馏体展示什么？隐藏什么？

---

## 产物格式
生成的内容会写入 `lib/persona.ts`，格式如下：

```ts
const SELF_MEMORY = `
## 技术能力记忆
...（你的真实能力描述）
## 项目经历
...（你的真实项目经历）
`;

const PERSONA = `
## 口头禅与语言风格
...（你的真实语言风格）
## 认知风格
...（你的真实决策模式）
## 社交模式
...（你的真实协作方式）
`;

const SAFETY_RULES = `
## 安全边界
...（你定义的不可输出内容）
`;

export const SYSTEM_PROMPT = `你是[你的代号]的数字蒸馏体...
${SELF_MEMORY}
${PERSONA}
${SAFETY_RULES}
`;
```
